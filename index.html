Phonebell

My friends and I made a website called Phonebell. Phonebell is a Web App that allows a user to scan a QR code at a door instead of ringing an old last-century doorbell. This QR code contains an encrypted url that interacts with the Twilio API to call or text the homeowner.

Technologies used: HTML, CSS, Javascript, Bootstrap, Python, Flask, Twilio API, ngrok
